# Planet icons

The five `starmap-planet-*` icons Factorio uses in its map-generator "Appears
on" column, downscaled to 64x64 from the game's own graphics:

| File | Source (Factorio 2.1.9 `Contents/data`) |
|------|------------------------------------------|
| `nauvis.png` | `base/graphics/icons/starmap-planet-nauvis.png` |
| `vulcanus.png` | `space-age/graphics/icons/starmap-planet-vulcanus.png` |
| `gleba.png` | `space-age/graphics/icons/starmap-planet-gleba.png` |
| `fulgora.png` | `space-age/graphics/icons/starmap-planet-fulgora.png` |
| `aquilo.png` | `space-age/graphics/icons/starmap-planet-aquilo.png` |

These are copyrighted assets owned by Wube Software, reused here for a
non-commercial fan tool. They are not covered by this project's license. If
redistribution ever becomes a concern, swap them for original stand-in icons
(the `PLANET_ICONS` map in `src/model/planets.ts` is the single swap point).
